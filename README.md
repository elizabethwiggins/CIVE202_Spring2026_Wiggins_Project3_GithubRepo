# CIVE202_Spring2026_Wiggins_Project3_GithubRepo
The Federal Highway Administration (FHWA) is looking for clean and simple graphs that showcase various trends and patterns in the dataset. Therefore, this project will be providing these graphs by using coding software to generate plots that are clear and purposeful. The traffic data featured in the following graphs is sourced from the National Household Travel Survey (NHTS) and the Next Generation Simulation (NGSIM) datasets.

This project concentrates on comparing different variables in the previously listed datasets and graphing them in a clear and efficient manner. This will allow for patterns in traffic to be determined and inferences to be made that allow for a better understanding of transportation and daily life. 
The program’s output will present 6 graphs that portray different patterns and trends in the traffic data. Graph types include a bar chart, histogram, boxplot, time-series, and more. One of the graphs output by the code is data generated from an intelligent driver model traffic simulation using NGSIM data. This graph will compare the acceleration vs time of one specified vehicle pair from the dataset to a simulated follower in the simulated traffic scenario. 
Final deliverables include: a functioning Jupyter Notebook, an annotated code documentation, a GitHub repository with a user guide, a Gantt chart, the Scope of Work, time sheet, and a comprehensive technical report.

User Guide: Simply open the Jupyter notebook with all the code and run each box of code to present graph results.
